PR

Project Resolver

Walks the directory tree to identify project boundaries. Uses .git roots and pyproject.toml markers. Loads project-specific configuration from Subconscious.af.

EF

Eager File Generator

Always writes. If AGENTS.md doesn't exist, creates it from template. No prompts, no questions — just generates the file so the session can proceed immediately.

AG

AGENTS.md Injection

Parses AGENTS.md and prepends extracted rules to the system prompt. Provides context-aware instructions tailored to the specific project.

OH

Orphan Handler

Tracks all opened resources (files, connections, handles). Ensures every open() has a matching close(). Closes all resources on session end.

Memory Persistence Cycle

sequenceDiagram
    participant T as Terminal
    participant S as Kimi Session
    participant R as Project Resolver
    participant L as Letta Core
    participant F as Subconscious.af

    T->>S: Start session in ~/project
    activate S
    
    S->>R: Identify project
    activate R
    R-->>S: Project context
    deactivate R
    
    S->>L: Request memories
    activate L
    L-->>S: Relevant context
    deactivate L
    
    S->>S: Work on task...
    
    S->>L: Store Important Shit™
    activate L
    L->>F: Compress & write
    deactivate L
    
    S->>T: Close session
    deactivate S
